# CDN-Finder
A tool for detect which ip of cidr behind the CDN.


# Installation

```
go install github.com/SharokhAtaie/cdn-finder@latest
```


# Usage

```
cdn-finder -list ips.txt
cat ips.txt | cdn-finder
```
